# Code of Conduct

## Our Commitment
We are committed to making this project welcoming and inclusive.

## Expected Behavior
- Be respectful to others
- Use welcoming and inclusive language
- Provide constructive feedback

## Unacceptable Behavior
- Insults or personal attacks
- Discriminatory comments
- Harassment or threats

## Reporting Issues
If someone violates this Code of Conduct, please open an issue or contact the project maintainers.

## Enforcement
Project maintainers may take appropriate action if the Code of Conduct is violated.
